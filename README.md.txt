# Executive Alpha | Institutional Master Terminal

**Executive Alpha** es una terminal financiera de alto rendimiento diseñada para el análisis de riesgo y la gestión de flujos de caja (Cashflow) en entornos institucionales.

## 🚀 Características
- **Matrix de Activos:** Monitoreo de renta variable, ETFs y Criptoactivos.
- **Escudo de Riesgo:** Calculadora avanzada de lotaje basada en la distancia del Stop Loss.
- **Motor de Apalancamiento:** Simulación de poder de compra y niveles de liquidación.
- **Gestión de Cashflow:** Proyección de rendimientos netos con cálculo automático de impuestos (5%).
- **Market Insights:** Análisis cuantitativo integrado para la toma de decisiones.

## 🛠️ Tecnología
- **Frontend:** HTML5, CSS3 (Tailwind CSS).
- **Engine:** JavaScript Vanilla (ES6+).
- **Monetización:** Google AdSense Ready.
